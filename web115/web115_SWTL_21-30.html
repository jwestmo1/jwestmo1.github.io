Chapter 21

Changing Case - Changes an entry to all upper or lower case

Syntax - cityToCheck = cityToCheck.toLowerCase(); - Then you have apples to apples

Syntax - cityToCheck = cityToCheck.toUpperCase(); - Then you have apples to apples

Chapter 22

Strings: Measuring length and extracting parts - Using slice to get parts of input

Syntax - var firstChar = cityToCheck.slice(0,1); - leaves cityToCheck alone but removes first letter. first number in () is the first letter in the index and the 1 is the number to start out so (0,1) gets you the first letter

Syntax - var otherChar = cityToCheck.slice(1); - One shows you where in the index to start and nothing else is specified so it goes to the end

Syntax - then you can turn the first letter into upper-case and the rest into lower-case ad concat the two together under a new var to have the casing correct.

Chapter 23

Strings: Finding segments - How to locate a string segment(phrase)

Syntax - var firstChar = text.indexOf("World War II"); - Looks for anf finds the first instance of this phrase so you can replace it with another. Can make a loop to keep replacing multiple times.

Syntax - text.lastIndexOf - Finds last instance of string

Chapter 24

Strings: Finding a character at a location - How to find the first character

Syntax - firstName.slice(0,1) works and so does firstName.charAt(0);

Syntax - var lastChar = name.charAt(name.length - 1); Finds the last charcater in a dtring. charAt can only ID a character. It can't change it.

Chapter 25

Strings: Replacing characters - replace method

Syntax - var newText = text.replce("World War II", "the Second World War"); first string is what is to be replaced and the second is what to replace it with. Only for first instance

Syntax - global replace - var newText = text.replce(/World War II/g, "the Second World War"); Replace quotes on phrase to replace with / and add a g after the 2nd one

Chapter 26

Rounding Numbers - var numOfStars = Math.round(scoreAvg); - Math must be capitalized. Can assign to new var or replace old variable

Syntax - var scoreAvg = Math.ceil(0.000001); = 1 - This rounds up no matter how small the decimal

Syntax - var scoreAvg = Math.floor(0.99999); = 0 - This rounds down no matter how big the decimal

Chapter 27

Generating Random Numbers - generates random number like a simulation of a dice throw

Syntax - var randomNumber = Math.random(); - Generates a 16-place decimal number. If you need 6 possible numbers numltiple by 6 and then add 1. Then use floor to round down.

Chapter 28

Converting strings to integers and decimals - Sometimes JS turns a string temporarily into a number to be used for math. Here's how to always convert it.

Syntax - var currentAge = prompt("Enter your age.");

Syntax - var qualifyingAge = parseInt(currentAge) + 1; - takes your entry, truns it into a number and adds 1.

Other Info - if you had a number 1.999 and you use parseInt - it will give you 1. It does not round. If you want to preserve a decimal number, use parseFloat instead of parseInt. That would give you the entire 1.999

Chapter 29

Converting strings to numbers, numbers to strings - Use Number to turn a string into an int or float. Use .toString() to turn a number to a string

Syntax - var string = "24"; var num = Number(string);

Syntax - var num = 1234; var string = num.toString();

Chapter 30

Controlling the length of decimals - number of decimal places

Syntax - var total = 10.59675 - var prettyTotal = total.toFixed(2); - rounds to 2 decimal plces - 10.60
