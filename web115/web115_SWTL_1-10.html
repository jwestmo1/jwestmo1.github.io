Chapter 3

Declaring a number variable - how to declare a number variable

Syntax - var number = 25; - This declares the variable named number and assigns the number 25 to it.

Syntax - var newNum = number + 500; - This establishes a new variable and adds the first variable plus another number to it. The answer would be 525.

Chapter 4

Variable Names Legal and Illegal - what you can use to name a variable

Syntax - 1. Do not use a reserved word as a variable name. 2. Do not use just a number or start wuth a number for your variable. 3. Variables cannot contain spaces. 4. Variables can only contain letters, numbers, dollar signs, and underscore. 5. Variables are case sensitive and you should use camelCasing.
