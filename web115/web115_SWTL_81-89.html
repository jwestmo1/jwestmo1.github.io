Chapter 71

Objects: Methods - Attach a function to an object to become the objects method

Syntax - The word this is used instead of repeating the name of the object.

Syntax - Use this.whatever and JS knows you are referring to a property of the object being defined

Syntax - EX: var bestPrice = plan1.price OR var bestPrice = this.price

Chapter 72

Objects: Constructors - Constructor Function

Syntax - When you have lots of things to add to lots of objects you need a way to do so quickly. Construction functions have capitalized variables

Syntax - function Plan(name, price, space, transfer, pages) {

Syntax - this.name = name;

Syntax - this.price = price;

Syntax - this.space = space;

Syntax - this.transfer = trasfer;

Syntax - this.pages = pages;}

Chapter 73

Objects: Constructors for Methods - build the method as well as the properties into the constructor function

Syntax - When you attach a variable to an object, it is called a property of the object

Syntax - When you attach a function to an object, it is called a method of the object

Syntax - Using the above object add property that calls function: this.calcAnnual = function(percentIfDisc)

Syntax - Call it like this: var annualPrice = ps.calcAnnual(.85)

Chapter 74

Objects: Prototypes - One copy of the method shared by all objects in the contructor

Syntax - Don't make in the constructor function. Instead define with:

Syntax - Plan.prototype.calcAnnual = function(percentIfDisc)

Syntax - Rest of code is the same.

Syntax - Can have prototype properties too. Plan.prototype.cancellable = true;

Chapter 75

Objects: Checking for Properties and Methods - see if an object has a particular property.

Syntax - var gotTheProperty = "price" in plan1; = Checks if plan 1 has a property called price and returns boolean

Chapter 76

Browser Control: Getting and Setting the URL - find where the browser is at and do things

Syntax - Current location: var whereWeAt = window.location.href;

Syntax - Can get pieces: var theDomain = window.location.hostname;

Syntax - Can get host and path: var thePath = window.location.pathname;

Syntax - Can get the hash: var theAnchor = window.location.hash;

Syntax - Can use the pieces to go to different places in site. var destination = "http://" + theDomain + "wow.html";

Chapter 77

Browser Control: Getting and Setting the URL Another Way - asssigning and replaing the URL

Syntax - Assigning a URL with history intact window.location.assign("http://me.com")

Syntax - Assigning a URL and interfering with browser history - window.location.replace("http://me.com")

Syntax - To reload current page use - window.location.reload();

Chapter 78

Browser Control: Forward and Reverse - as if the user did backspace or forward on browser button

Syntax - Backward - history.back();

Syntax - Forward - history.forward();

Syntax - Can go back and forward using positive and negative numbers - history.go(-1); OR history.back()

Chapter 79

Browser Control: Filling the Window with Content- making popups

Syntax - You can't control where the window opens - var monkeyWindow = window.open();

Syntax - Use write method to fill window

Syntax - Close window with monkeyWindow.close();

Chapter 80

Browser Control: The Window's Size and Location- open a window with a URL

Syntax - var monkeyWindow = window.open("monk.html", "win1");

Syntax - With specific size window - var monkeyWindow = window.open("monk.html", "win1", "width=420,height=380"); - no spaces

Syntax - Where to open window on page - var monkeyWindow = window.open("monk.html", "win1", "width=420,height=380", left=200, top=100);
