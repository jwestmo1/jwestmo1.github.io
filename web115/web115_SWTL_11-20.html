Chapter 17

Arrays: Removing and Inserting Elements - shift, unshift, splice, and slice

Syntax - Shift - remove an element from the beginning of the array - Ex: dog, cat, bird. pets.shift(); list is now cat, bird

Syntax - Unshift - add an element to the beginning of the array - Ex: dog, cat, bird. pets.unshift("lizard", "fish"); list is now lizard, fish, dog, cat, bird

Syntax - Splice - add one or more elements anywhere in the array - Ex: dog, cat, bird. Add two animals and remove cat. pets.splice(1,1, "pig", "duck"); This makes your list dog, pig, duck, bird. First number is where to add, second number is how many to remove, the rest are what to add. You can add without removing and remove without adding

Syntax - Slice - copy on or more consecutive elements and put them in a new array and leaves the orginial array unchanged. Ex: dig, bird, cat, goat, pig, duck var noPets = pets.slice(2,4); this moves cat and goat into an array by itself. Use the index of where you want to start to copy and the index after your last one you want to copy.
